# augustin.ribreau.co

## 1️⃣ : clone
```bash
git clone https://github.com/AugustinRibreau/augustin.ribreau.co.git
```

## 2️⃣ : build
###### with parceljs ❤️
```bash
npm install
npm run build
```

## 3️⃣ : start
```bash
npm run start:dev
```

## How to contribute
Just send a pull-request 😉
